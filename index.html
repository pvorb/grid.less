    • grid.less

    a semantic grid framework for less css

    Project maintained by pvorb Hosted on GitHub Pages — Theme by mattgraham

    This framework lets you configure your personal grid.

    Getting Started

    1. Install Less (npm install -g less).

    2. Download grid.less.

    3. Start your new less stylesheet (e.g. style.less) and specify the basic variables.

       @import 'grid.less';

 @cols: 12;
 @colwidth: 60px;
 @colspace: 20px;

    4. Compile with lessc style.less > style.css.

    5. Set up you basic HTML document (e.g. index.html).

       <!DOCTYPE html>
 <html>
   <head>
     <meta charset="utf-8">
     <title>grid.less test</title>
     <link rel="stylesheet" href="style.css">
   </head>
   <body>
     <header><h1>grid.less test</h1></header>
     <article>
       <h2>A test article</h2>
       <p>Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua. At vero eos et accusam et justo duo dolores et ea rebum. Stet clita kasd gubergren, no sea takimata sanctus est Lorem ipsum dolor sit amet. Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua. At vero eos et accusam et justo duo dolores et ea rebum. Stet clita kasd gubergren, no sea takimata sanctus est Lorem ipsum dolor sit amet.<p>
     </article>
     <aside>
       <p>Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua. At vero eos et accusam et justo duo dolores et ea rebum.</p>
     </aside>
     <footer>Copyright &copy; 2012 Jon Doe</footer>
   </body>
 </html>

    6. Now extend style.less with

       // ...

 header {
   .span(12); // span the whole width
 }
 article {
   .br; // break to a new row
   .span(8); // span 8 columns
 }
 aside {
   .span(4); // span 4 columns
 }
 footer {
    .br; // break to a new row
    .skip(4); // skip 4 columns
    .span(8); // span 8 columns
 }

    7. Open index.html in your favorite browser and look at the result.
